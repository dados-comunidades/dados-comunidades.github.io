Data Communities

A web mapping project to help brazilians explore and explain their community to the world.

Planning an App:

Here is the planned architecture for the app. Progressive, responsive and almost 100% JavaScript

Server: Node JS Express JS
Front end: Angular 2 Bootstrap CSS
Mapping: Leaflet JS Turf JS
Storage: Firebase
